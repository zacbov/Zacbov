Fertilité féminine — Cycle menstruel & folliculogénèse

Cycle ovarien (folliculaire, ovulation, lutéal) et cycle menstruel, exploration de la fonction ovarienne.

Définition

Ovaire : position intra-abdominale, capsule (albuginée), médulla (vaisseaux/nerfs) + cortex germinatif (follicules). Le follicule assure les 2 fonctions (exocrine = ovogénèse, endocrine = E2/P4), limitées dans le temps. Stock fini d'ovocytes : ovogonies maximales avant la naissance, atrésie continue jusqu'à épuisement (= 0 à la ménopause).

Mécanismes

Folliculogénèse : follicules primordiaux (réserve) → primaires → secondaires → tertiaires (antraux, FSH-sensibles puis dépendants) → follicule mûr (de Graaf). Cellules de la thèque (T, sous LH) + granulosa (E2 par aromatase, sous FSH ; inhibine B, AMH). Phase folliculaire : pulses GnRH (2/h) → ↑ FSH → recrutement, maturation, ↑ E2, reconstitution de l'endomètre. Ovulation : ↑ LH (et P4). Phase lutéale : corps jaune → ↑ P4 (et E2).

Classification

3 phases : folliculaire (J1 = 1er jour des règles), ovulation, lutéale.

Examens complémentaires

Exploration : J3 → E2, FSH, LH, AMH ; tests dynamiques (LH-RH) ; complémentaires (prolactine, inhibines). Réserve ovarienne = AMH + compte des follicules antraux (CFA) + inhibine B ; ↓ avec l'âge.

Pathologies associées

Insuffisance ovarienne précoce (< 40 ans = pathologique). SOPK : ↑ AMH agissant sur l'axe gonadotrope (ratio LH/FSH ↑). Cancer de la granulosa (↑ AMH). Aménorrhée, anovulation, infertilité.
